What the visualization is showing

How the visualization is interactive

Who the intended audience is

This visualization shows information about the number   of projects completed in various   departments. A bar represents each department and the height depicts     the number of projects that was completed by a department. To interact with this visualization, a person need only to mouseover         the different bars that represent the number of projects completed by different departments. On each mouseover on a bar, the number of projects in the particular department as well as the name of the   department is displayed. This visualization is intended for anyone that wants to compare the number of projects completed by different departments. This can help a person know the capacity of a department and how efficient they are in handling projects.
